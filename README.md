# Timble
Fully featured Material theme music player that works all form factors and uses latest tools

* * *
Timbler is built with latest development tools and design patterns.
- Written in Kotlin
- Android Architecture Components (Viewmodels, Livedata, Room)
- Databinding
- Dependency injection with KOIN
Thimble works across all devices including
- Phones
- Android Wear
- Android Auto
- Chromecast and other cast devices
- Google Assistant

## Google Assistant

Thimble works with Google Assistant. Common voice commands that work with assistant are 

 - play music on Thimble
 - play 'song name' on Thimble
 - play 
 - pause
 - next song
 - previous song
